COMMIT

Video, 1' 16", 2018

COMMIT is a data visualization and sonification work created by Ren Yuan. By getting the commit history of more than 200 developers on the p5.js project on GitHub, visualize and audible developers to add, modify, delete, and rename project files in a daily update manner. The commit history is from February 2013 to October 2018.
