SORTING

Video, 1' 50", 2018

SORTING is an algorithm visualization and sonification work created by Ren Yuan. Through computer programming, the sorting history, data comparison, data swap, absolute error, residual sum of squares and other features are visualized and audible by seven sorting algorithms (insertion sort, shell sort, bubble sort, quick sort, selection sort, heap sort, merge sort). The problem size of all sorting algorithms is 31, and the initial condition is random.
